# 提示词编写指南

本文档提供编写优质图片生成提示词的技巧和最佳实践。

## 1. 基本原则

### 1.1 核心要素

一个优秀的提示词应包含以下要素:

1. **主体** - 图片的主要内容
2. **环境/场景** - 背景和周围环境
3. **风格** - 艺术风格或视觉风格
4. **光线** - 光照效果
5. **视角** - 拍摄角度
6. **质量** - 画质细节

### 1.2 语言选择

- **Qwen**: 中英文皆可,中文效果优秀,尤其擅长中文文本渲染
- **Gemini**: 推荐英文提示词,英文理解更准确

## 2. 提示词结构

### 2.1 标准模板

```
[主体] + [动作/姿态] + [环境] + [风格] + [光线] + [质量]
```

#### 示例

```
一只橘黄色的猫 + 坐在沙发上 + 温馨的客厅 + 电影感 + 温暖的阳光 + 4K高清
```

完整提示词:
```
一只橘黄色的猫坐在沙发上,温馨的客厅,电影感,温暖的阳光从窗户照进来,4K高清,细节丰富
```

### 2.2 拆解示例

**提示词**: "一位年轻女性站在巴黎埃菲尔铁塔下,傍晚,金色夕阳,浪漫风格,专业摄影,8K分辨率"

**拆解**:
- 主体: 年轻女性
- 动作: 站立
- 环境: 巴黎埃菲尔铁塔下
- 时间: 傍晚
- 风格: 浪漫风格,专业摄影
- 光线: 金色夕阳
- 质量: 8K分辨率

## 5. 负向提示词

负向提示词用于排除不想要的效果,仅在 Qwen 中支持。

### 5.1 常用负向提示词

```
低分辨率,模糊,低画质,畸形,手指畸形,肢体畸形,多余手指,文字错误,水印,签名,截断,构图不佳,光线不足
```

### 5.2 按类别分类

#### 质量问题
```
低分辨率,模糊,低画质,颗粒感,压缩痕迹
```

#### 人体问题
```
畸形,手指畸形,肢体畸形,多余手指,多余肢体,面部扭曲,眼睛不对称
```

#### 文字问题
```
文字错误,乱码,字体扭曲,排版错误
```

#### 构图问题
```
构图不佳,主体不突出,边缘截断,背景杂乱
```

#### 光线问题
```
光线不足,曝光过度,光线不自然,阴影过重
```

### 5.3 使用示例

```python
negative_prompt = "低分辨率,模糊,低画质,畸形,手指畸形,肢体畸形,多余手指,文字错误,水印,签名"
```

## 6. 高级技巧

### 6.1 提示词扩展

开启 `prompt_extend` 参数,让 AI 自动优化和扩展提示词:

```python
# Qwen
parameters = {
    "prompt_extend": True  # 默认 True
}
```

### 6.2 使用随机种子

使用固定种子可以复现相同的生成结果:

```python
# Qwen
parameters = {
    "seed": 12345  # 固定种子
}
```

### 6.3 多版本生成

生成多个版本,选择最满意的结果:

```python
# Qwen 生成 4 张
parameters = {
    "n": 4
}
```

### 6.4 迭代优化

1. 使用基础提示词生成图片
2. 根据结果调整提示词
3. 添加细节描述或修改风格
4. 重新生成,逐步优化

### 6.5 参考借鉴

- 查看 AI 艺术社区的优秀提示词
- 参考 Midjourney、Stable Diffusion 的提示词库
- 学习摄影、绘画作品的描述方式

## 7. 常见错误

### 7.1 描述过于简单

❌ 错误: "一只猫"

✅ 正确: "一只橘黄色的猫坐在沙发上,阳光照进来,温馨的氛围,4K高清"

### 7.2 缺少风格描述

❌ 错误: "一位女性"

✅ 正确: "一位年轻女性肖像,写实风格,电影感,专业摄影"

### 7.3 中英文混用

❌ 错误: "一只猫, cute, 温馨"

✅ 正确: "一只可爱的猫,温馨的氛围" (Qwen)

✅ 正确: "A cute cat, warm and cozy" (Gemini)

### 7.4 提示词过长

Qwen 限制 800 字符,Gemini 建议控制在合理范围内。

### 7.5 忽略负向提示词

❌ 错误: 生成人物图片时不设置负向提示词

✅ 正确: 设置负向提示词排除畸形和低质量问题

## 8. 不同场景的提示词模板

### 8.1 社交媒体配图

```
主体 + 动作 + 简洁背景 + 鲜艳色彩 + 现代风格 + 高清
```

示例:
```
一杯拿铁咖啡,放在白色桌面上,背景是模糊的城市街道,晨光,现代简约风格,Instagram 风格
```

### 8.2 博客封面

```
抽象主体 + 渐变背景 + 大字(仅 Qwen) + 现代设计 + 4K
```

示例:
```
抽象的科技感图形,蓝色到紫色渐变背景,中央大字"AI 未来",现代设计风格,科技感,4K高清
```

### 8.3 产品展示

```
产品 + 使用场景 + 自然光 + 专业摄影 + 浅景深 + 高清
```

示例:
```
智能手机特写,放在木质桌面上,旁边有咖啡杯,自然窗光,专业产品摄影,浅景深,8K分辨率
```

### 8.4 概念艺术

```
幻想主体 + 奇幻场景 + 戏剧性光线 + 艺术风格 + 细节丰富
```

示例:
```
漂浮在云端的城堡,巨大的翅膀建筑结构,金色夕阳,奇幻风格,油画质感,细节丰富
```
